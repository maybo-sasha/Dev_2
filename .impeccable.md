# Impeccable — Project Design Context

## Design Context

### Users
Prospective employers, clients, recruiters and fellow designers viewing a personal
design portfolio. They arrive to quickly judge craft, taste and range — motion, 3D,
game UI and product design — often skimming on desktop or mobile. The job to be done:
"convince me, fast, that this person has exceptional visual + interaction craft."

### Brand Personality
Confident, crafted, playful. Visual-first voice — the work speaks, copy supports.
Should evoke confidence and delight without feeling corporate or templated.

### Aesthetic Direction
Dark, cinematic and motion-rich, built on a shared chrome (persistent nav, custom
cursor on the home page, smooth Lenis scrolling, GSAP transitions). Each project page
gets its **own world / palette** inside that chrome:
- **Home** — near-black with a WebGL particle field; custom expanding cursor.
- **Lucky Buddies** — deep navy `#0a0e1e` + orange/yellow, bold uppercase display.
- **Playground** — charcoal `#0b0b0f` + violet/acid-lime "creative lab", mono labels.
- **Tetris Block Party** — dark `#0a0e1e` + Tetris-blue/rainbow accents, zebra sections.

References: modern design-studio / award-style case-study sites.
Anti-references: generic corporate templates, cluttered dashboards, stocky AI-gen looks.

### Typography
- **Display:** `HelveticaNeue-Roman` (Helvetica Neue) via `--font-display` — clean,
  neutral Swiss headings; large, tight, confident.
- **Body:** `Poppins` via `--font-body` — comfortable line-height for paragraphs.
- **Mono accents:** technical labels on Playground use a monospace stack (`--font-mono`).
- Headings use `text-wrap: balance`; paragraphs use `text-wrap: pretty`.

### Design Principles
1. **Motion with purpose.** Smooth, strong `ease-out` (`cubic-bezier(0.23,1,0.32,1)`),
   short stagger (~60ms), snappy hovers; honor `prefers-reduced-motion`. Never animate
   for its own sake or on repeated/keyboard actions.
2. **Each project is its own world.** A distinct palette + personality per case study,
   unified by the shared nav, cursor, page transitions and type system.
3. **Bold typographic hierarchy.** Big, confident grotesk headings; generous whitespace;
   clear primary/secondary contrast.
4. **Show, don't tell.** Video and imagery lead; copy is concise and supportive.
5. **Details compound.** Consistent tokens (easing, color vars, spacing), correct
   transform-origins, no faux-bold, no dead imports — the invisible correctness adds up.
