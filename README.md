## About

An interactive portfolio built with smooth page transitions, WebGL particle backgrounds, custom cursor interactions, and scroll-driven motion. Each project page showcases case-study work with video, 3D, and rich visual detail.

## Tech stack

| Layer | Tools |
|-------|-------|
| Build | [Vite](https://vitejs.dev/) |
| 3D & WebGL | [Three.js](https://threejs.org/), postprocessing |
| Motion | [GSAP](https://gsap.com/), ScrollMagic |
| Scroll | [Lenis](https://github.com/studio-freight/lenis), Locomotive Scroll |
| Routing | [Barba.js](https://barba.js.org/) |

## Local development

```bash
# Install dependencies
npm install

# Start dev server at http://localhost:5173
npm start

# Production build (outputs to dist/)
npm run build
```
